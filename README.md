# Poll App

## Dependency
* Node version 15+ is recommended
* yarn

## How to run the app
create-react-app is used to build this app.

* clone the repo.
* navigate to the cloned repo and install dependencies using the command `yarn install`.
* start the app by command `yarn start`.
